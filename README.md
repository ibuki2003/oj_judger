# Judger
## 概要
[OnlineJudge](/ibuki2003/onlinejudge2) に対応したジャッジプログラムです.
Python3にてご利用ください.


## 環境
- Python3
    - < 3.4
- PyMySQL

## 初期設定
1. config.iniをコピーしてくる
2. 環境に合わせ書き換え
3. main.pyを実行,常駐

## 注意
- ジャッジが動いているときにC-cで停止をしないでください.
    - **仕様です**
    - 子プロセスのみを終了することとなりめんどくさいことになるので
- 終了時にエラーが出ることがあります
    - **仕様です**
